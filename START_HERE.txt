 🚀 QUICK START

  1. Setup MongoDB (2 min)
     → mongodb.com/cloud/atlas
     → Create free cluster
     → Copy connection string

  2. Git push (1 min)
     → cd /path/to/site
     → git add .
     → git commit -m "Deploy to Vercel + Render"
     → git push origin main

  3. Deploy Backend (7 min)
     → render.com
     → New > Blueprint > Select repo
     → Add MONGODB_URI env var
     → Deploy

  4. Deploy Frontend (3 min)
     → vercel.com
     → Add Project > Select repo
     → Root: frontend | Framework: Vite
     → Add VITE_API_URL env var
     → Deploy

  5. Finalize (1 min)
     → Render > Backend > Update FRONTEND_URL
     → Manual Deploy

  6. Test (2 min)
     → Open YOUR-APP.vercel.app
     → Try signup/login
     → Check Network tab for errors
