🎯 YOUR 15-20 MINUTE PATH TO LIVE
═══════════════════════════════════════════════════════════════════════════

Step 1: MongoDB (2 min)
  ├─ Go to mongodb.com/cloud/atlas
  ├─ Create free cluster
  ├─ Get connection string
  └─ Save it

Step 2: GitHub (1 min, if needed)
  ├─ Create GitHub account
  ├─ Create new repository
  └─ Push your code

Step 3: Render Backend (7-10 min)
  ├─ Go to render.com
  ├─ Connect GitHub
  ├─ Add MongoDB connection string
  └─ Click Deploy

Step 4: Vercel Frontend (2-3 min)
  ├─ Go to vercel.com
  ├─ Connect GitHub
  ├─ Add backend URL from Render
  └─ Click Deploy

Step 5: Finalize (1 min)
  ├─ Update backend's FRONTEND_URL with Vercel URL
  └─ Manual deploy in Render

Step 6: Test (2 min)
  ├─ Open https://YOUR-APP.vercel.app
  ├─ Try signup/login
  └─ Celebrate! 🎉
